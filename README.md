# 🚀 AI Based Sales Forecasting

An AI-powered Sales Forecasting Web Application built using Machine Learning and Streamlit.

## 📌 Features

- 📈 AI Sales Prediction
- 📂 CSV Dataset Upload
- 📊 Interactive Dashboard
- 🎨 Professional UI Design
- 📉 Animated Charts & Graphs
- ☁️ Streamlit Cloud Deployment

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
